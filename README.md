# :computer: Game - Fundamentos de Programação :computer:
  
  
## :bookmark: Sobre
O jogo foi desenvolvido para a disciplina de Fundamentos de Programação e tem como proposta principal ser um jogo educativo para crianças em período de alfabetização.


## :construction: Desenvolvimento
O jogo foi todo desenvolvido usando Python e a biblioteca [Pygame](https://www.pygame.org/news), as imagens utilizadas foram coletadas da internet e a música de fundo foi baixada do site [Bensound](https://www.bensound.com/). A máquina que irá executar não precisa ter Python instalado, apenas o [C++](https://support.microsoft.com/pt-br/help/2977003/the-latest-supported-visual-c-downloads).


## :video_game: Gameplay
No jogo o usuário irá controlar um livro mágico :green_book: que deve coletar todas as letras do alfabeto sem ser atingido pelo lápis :pencil2:.
